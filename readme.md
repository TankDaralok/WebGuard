# WebGuard Global - Advanced URL Verification Tool

A Flask-based web application that analyzes URLs for trustworthiness using multiple technical indicators.

## Features
- **Global Reach:** Supports English, Hungarian, German, and Spanish.
- **Deep Analysis:** Checks SSL status, Domain Age (WHOIS), MX Records (Email), and Typosquatting.
- **Scam Detection:** Analyzes HTML content for high-pressure scam keywords.
